# README.md
# 🏃 퍼즐형 빙고 러닝 (Winter 4-Week)

## 한 줄 소개
**빙고판을 채우는 게임이 아니라, 내가 고른 카드로 “내 빙고판”을 만들고 완주하는 러닝 게임**입니다.  
기본은 **1회 러닝 = 최소 1칸**, 조건/협동이 겹치면 **2칸**, 매우 희귀하게 **3칸**까지 가능합니다.

---

## 기본 정보
- 기간: 4주(겨울 시즌)  
- 목표: 개인 빙고판 **25칸 완주**
- 핵심 러닝 축  
  - **목요미식회(고정 인터벌)**: 매주 목요일  
  - **벙개 러닝(자율)**: 상시(누구나 개설/참여)

---

## 시작하기(Quick Start)
1) **내 티어 확인**: 초보 / 중수 / 고수  
2) **25장 카드 선택** (타입별 고정 구성)
   - Base(A) 10장 + Condition(B) 7장 + Co-op(C) 5장 + Marathon(D) 2장 + Wild(W) 1장(중앙)
   - *시즌 변형 룰(공지 시): 중수 W 2장(중앙+모서리1), 고수 W 3장(중앙+모서리2)*
   - *A(Base) 카드는 티어 스케일링 적용(카드덱 `조건(티어 스케일링)` 참고)*
3) **빙고판 배치**
   - 중앙칸: **W 전용**
   - C(협동) 카드: 서로 **상하좌우 인접 배치 금지**
   - D(마라톤) 카드: **모서리 배치 금지**
4) 게임 기간 동안 러닝 인증 → 칸 색칠

---

## 한 번의 러닝으로 몇 칸까지?
- **러닝 기록(앱 활동 1개)** 기준으로  
  - 체크 가능한 칸: **최대 3칸**
  - 같은 러닝에서
    - Base(A) 최대 1칸
    - Condition(B) 최대 1칸
    - Co-op(C) 최대 1칸
  - (D/W가 해당 러닝에서 완성되는 경우에도 **총 3칸 제한**)

---

## Wild(W) & Wild 토큰 규칙
- W 카드를 달성하여 체크하면 **Wild 토큰 1개**를 획득합니다(중앙/모서리 W 모두, 보유 최대 1개).
- Wild 토큰은 1회 사용(택1)
  - **공격(Seal)**: 상대의 특정 타입(B 또는 C)을 잠시 봉인  
  - **방어(Shield)**: 내가 Seal 대상이 되면 1회 무효화

자세한 규칙은 `RuleBook.md` 참고.

## 부문별 우승(승리 조건)
- 최단기간 **5빙고**
- **올빙고(25칸 완주)**
- **최다 별(★) 수집**

자세한 기준/동률 처리는 `RuleBook.md` 참고.

## 인증(기본)
- 러닝 1회마다:
  - 러닝앱 캡처(거리/시간/날짜) 또는 링크
  - 해당 카드 요구사항(날씨/동료/장소 등) 충족 증거(간단 캡처/사진/태그)
- 운영 방식/채널은 운영진 공지에 따릅니다.
