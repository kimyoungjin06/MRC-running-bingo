# 🏃 퍼즐형 빙고 러닝 (간략)

## 핵심만 5줄
- 42장 카드 중 **25장 선택** → 나만의 **5×5 빙고판** 제작
- 러닝 1회당 **최대 2칸 체크** (A/B/C 중 2칸, 타입별 1칸 제한)
- D/W는 **완성되는 러닝에서 체크**
- W 달성 시 **Wild 토큰 1개** (공격/방어 1회)
- 우승 부문: **최단 5빙고 / 올빙고 / 최다 별(★)**

---

## 시작 3단계
1) **티어 선택** (초보/중수/고수)  
2) **25장 고르기** (A/B/C/D/W 타입 수량 맞추기)  
3) **배치 규칙**: 중앙 W, C 인접 금지, D 모서리 금지

---

## 러닝 체크 규칙
- **1회 러닝 = 최대 2칸**
- A/B/C 중 최대 2칸 선택(타입별 1칸 제한)
- 여러 조건을 만족해도 **최대 2칸만 선택**

---

## 인증(최소)
- 러닝앱 캡처(거리/시간/날짜)
- 조건/협동 증거(사진/태그 등)
