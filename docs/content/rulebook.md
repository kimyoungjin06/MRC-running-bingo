# 📘 퍼즐형 빙고 러닝 룰북 v1.0

## 0. 목표/철학
- 초/중/고수 모두 **25회 러닝으로 25칸 완주 가능**하도록 설계
- 기본은 1회 1칸, 겹치면 2칸, 아주 희귀하게 2칸
- “잘 달리는 사람”보다 **판을 잘 운영(협동/조건/타이밍)**하는 사람이 유리

---

## 1. 용어 정의
- **러닝 기록(1회)**: Strava 등 러닝앱에 저장된 **활동 1개**(한 날 여러 활동이면 각각 1회로 계산)
- **칸(카드)**: 내가 선택해 빙고판에 올린 카드 1장(각 카드는 1회만 체크)
- **체크(달성)**: 조건을 만족한 카드 칸에 색칠하는 행위

---

## 2. 게임 준비(세팅)

### 2.1 카드 선택(드래프트/선택 방식)
- 기본 방식: **동시 선택(비공개 제출)**  
  참가자가 카드 코드만 보고 25장을 선택해 제출 → 운영진 검증 후 확정
- 타입별 고정 구성(권장, 25칸 고정)
  - Base(A) 10장
  - Condition(B) 7장
  - Co-op(C) 5장
  - Marathon(D) 2장
  - Wild(W) 1장
  - *변형 룰(선택, 시즌 사전 공지 필수)*  
    - 중수 전용: **W 2장(중앙 1 + 모서리 1)**, 예시 구성 A10 / B7 / C5 / D1 / W2(총 25장)  
    - 고수 전용: **W 3장(중앙 1 + 모서리 2) 필수 배치**, 예시 구성 A9 / B6 / C5 / D2 / W3(총 25장)  
    - 변형 룰 사용 시, W 배치 규칙  
      - 중수(W2): 중앙 1칸 + **모서리 1칸(좌상/우상/좌하/우하 중 선택)**  
      - 고수(W3): 중앙 1칸 + **모서리 2칸(좌상+우하 또는 우상+좌하)**

> W 카드는 “일정/성향”에 따라 달성 난이도가 갈리므로 **선택 전에 내용 공개**를 권장합니다.
> A(Base) 카드는 **티어 스케일링**이 적용됩니다(카드덱의 `조건(티어 스케일링)` 기준, 제목의 km/분 표기는 “중수 기준”).

### 2.2 빙고판 배치 규칙(필수)
- 중앙칸은 **W 전용**
- C(협동) 카드는 서로 **상하좌우 인접 배치 금지**(대각선 OK)
- D(마라톤) 카드는 **모서리 배치 금지**

---

## 3. 플레이 규칙(달성/중첩/상한)

### 3.1 러닝 1회당 체크 상한(가장 중요)
- 기본 러닝 인정 기준: **페이스 7'00/km 이내 + (티어별 거리 또는 시간 기준 충족)**
- 티어 기준: 초보 5km+ 또는 30분+ / 중수 7km+ 또는 40분+ / 고수 10km+ 또는 50분+
- 중수/고수 예외: **페이싱/레벨믹스 + 하위 티어 동행(그룹 티어 입력)**이면 기본 조건 예외 인정
- 러닝 기록 1개로 체크 가능한 칸: **최대 2칸**
- 같은 러닝에서 체크 가능한 타입 상한:
  - Base(A) 최대 1칸
  - Condition(B) 최대 1칸
  - Co-op(C) 최대 1칸
- 한 러닝이 여러 Condition(B)을 만족하더라도 **B는 1개만 선택**
- 한 러닝이 여러 Co-op(C)을 만족하더라도 **C는 1개만 선택**
- D/W가 “그 러닝으로 완성되는 경우”에도 **총 2칸 제한**
  - 예: A+B+C를 체크하려는 러닝에서 W까지 완성 조건이 동시에 충족되면 → 4개 중 2개만 선택 체크
- 3개 이상 충족 시: **참가자가 원하는 2개를 인증 시점에 선택하고 확정**(이후 변경 불가, 우선순위 강제 없음)

### 3.2 D(마라톤) 카드 체크 시점
- D 카드는 여러 날 누적 조건이며, 조건이 충족된 **마지막 러닝 기록**을 기준으로 완료 처리합니다.
- 완료 처리된 D는 해당 러닝에서 체크할 2칸 중 하나로 선택합니다(총 2칸 제한 준수).

## 4. Wild(W) & Wild 토큰 규칙

### 4.1 Wild 토큰 획득
- W카드를 달성하여 체크하는 순간, **Wild 토큰 1개** 획득(중앙/모서리 W 모두)
- 토큰 보유 제한: **초보 1 / 중수 2 / 고수 3**
- 보유 상한을 넘는 추가 획득은 저장되지 않음
- 토큰을 사용해 보유 수가 줄면, 이후 W 달성으로 다시 채울 수 있음

### 4.2 Wild 토큰 사용(택1, 1회 소모)
- **공격(Seal)** 또는 **방어(Shield)** 중 택1로 사용

#### A) 공격: Seal(타입 봉인)
- 대상: 아래 조건을 모두 만족해야 함  
  1) **나보다 진행도(완료 칸 수)가 더 많은 사람** 또는 **전체 상위권(운영진이 정한 기준, 예: Top3)**  
  2) 동일 대상에게는 **시즌 중 1회만** Seal 가능
  3) 발동 시점 최신 리더보드 기준, 동률도 가능
- 봉인 타입: **B(Condition) 또는 C(Co-op) 중 1개 선택**  
  - A(Base) / W(Wild) 봉인 금지
- 기간: **대상의 다음 2회 러닝 기록 동안(서로 다른 날짜 기준, 같은 날 여러 번은 1회로 계산)**
- **동시 봉인 허용**: B/C 서로 다른 타입은 동시에 가능  
- **중복 봉인 금지**: 같은 타입 봉인은 진행 중일 때 추가로 걸 수 없음
- 효과: 봉인 기간 동안 해당 타입 카드는 **체크 불가**
  - 러닝/인증 자체는 정상 인정
  - 봉인 기간에 충족한 조건을 **나중에 소급 체크하는 것은 불가**(봉인의 의미 유지)

#### B) 방어: Shield(봉인 무효화)
- 내가 Seal 대상이 되었을 때, 토큰을 사용해 **해당 Seal 1회 무효화**
- 동시 봉인(B/C)이 있을 경우, **선택한 타입 1개만** 무효화
- 방어 성공 시:
  - Seal은 적용되지 않음
  - 방어 토큰은 소모됨  
  - (권장) 공격자의 토큰도 소모 처리

### 4.3 선언 방식(필수)
- Seal/Shield는 **공개 선언**해야 유효
- 권장 포맷:
  - `[SEAL] @대상 / 봉인: B타입 / 기간: 다음 2회 러닝`
  - `[SHIELD] @대상 SEAL 방어 사용(무효화)`

## 5. 부문상(승리 조건)
이 게임은 “1등 1명”이 아니라 **부문별 우승**으로 진행합니다.

### 5.1 최단기간 5빙고
- **빙고 라인**: 가로/세로/대각선 5칸 완성 1줄
- **5빙고**: 빙고 라인 5줄 완성
- 우승: 시즌 중 **5빙고를 가장 먼저 달성**한 참가자
- 인증: 보드 캡처 + `[BINGO]`로 선언(운영진/자동판정 기록)
- 동률 처리(권장): ① 인증 시간 ② 별 합계 ③ 완료 칸 수 ④ 운영진 판정

### 5.2 올빙고(25칸 완주)
- 우승: 시즌 중 **25칸을 가장 먼저 완주**한 참가자
- 올빙고 달성자가 없으면: 해당 부문 **우승자 없음**
- 동률 처리(권장): ① 인증 시간 ② 별 합계 ③ 운영진 판정

### 5.3 최다 별 수집
- 별(★)은 카드 난이도이며, **체크한 카드의 별 합계**를 점수로 사용
- 우승: 시즌 종료 시점(마감) 기준 **별 합계 1등**
- 동률 처리(권장): ① 완료 칸 수 ② 5빙고 달성 시점 ③ 운영진 판정

---

## 6. 인증/판정(최소 규칙)
- 기본: 러닝앱 캡처(거리/시간/날짜) + 필요한 추가 증거(사진/태그/날씨 캡처)
- 2칸 체크, W 달성, Seal/Shield는 **운영진 1명 확인(이모지 ✅ 등)** 권장
- 안전 우선: 날씨/노면이 위험하면 해당 미션은 생략 가능(대체/실내 러닝 활용)

## 7. 분쟁/예외
- 애매한 케이스는 “게임의 재미와 공정성”을 기준으로 운영진이 최종 판정
- 무리한 인증 강요 금지 / 개인정보 노출 최소화
